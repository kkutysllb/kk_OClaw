# UI Polish Skill

## Purpose

Improve usability and visual consistency without distracting from core workflow.

## Workflow

1. Identify the primary workflow and remove friction from that path first.
2. Keep controls near the panel or object they affect.
3. Preserve existing design system components and density.
4. Check desktop and narrow widths for overflow or overlapping text.
5. Verify hover, disabled, loading, selected, and empty states.

## Review Checklist

- Toolbars stay in one row when space allows and degrade cleanly.
- Collapse controls are placed in the panel header they control.
- Text does not overflow buttons or compact cards.
- Empty states are concise and actionable.
- UI changes do not alter unrelated workspace behavior.
