# 🦌 关于 KKOCLAW

> **源于开源，回馈开源**

**KKOCLAW** 是一个社区驱动的超级智能体（SuperAgent）平台，能够进行研究、编码和创作。
借助沙箱、记忆、工具和技能系统，它可以处理从几分钟到几小时不等的各类复杂任务。

## 🙌 致谢

我们衷心感谢那些让 KKOCLAW 成为现实的开源项目和贡献者。我们真正站在巨人的肩膀上。

### 核心框架

- **[LangChain](https://github.com/langchain-ai/langchain)**：驱动我们 LLM 交互和链式调用的出色框架。
- **[LangGraph](https://github.com/langchain-ai/langgraph)**：实现复杂多智能体编排。
- **[Next.js](https://nextjs.org/)**: 构建 Web 应用的先进框架。

### UI 库

- **[Shadcn](https://ui.shadcn.com/)**: 驱动我们 UI 的极简组件库。
- **[SToneX](https://github.com/stonexer)**: 感谢他对逐 token 视觉效果的宝贵贡献。

这些杰出项目构成了 KKOCLAW 的基石，体现了开源协作的变革力量。
