# 安装 KKOCLAW Harness

<Callout type="info" emoji="📦">
  KKOCLAW Harness Python 包以 <code>kkoclaw</code>
  名称发布。
</Callout>

KKOCLAW Harness 是构建自己 Super Agent 系统的 Python SDK 和运行时基础。

如果你想在自己的产品或工作流中整合技能、记忆、工具、沙箱和子 Agent 等 Agent 能力，这正是你需要的 KKOCLAW 组件。

## 包名称

包名：

```bash
pip install kkoclaw
```

## 当前状态

KKOCLAW Harness 已发布并可用于生产环境。

## Harness 将提供什么

Harness 专为想要在 KKOCLAW 运行时模型之上构建自己 Agent 系统的开发者设计。

它将提供以下基础：

- 构建长时序 Agent
- 组合运行时能力（记忆、工具、技能、子 Agent）
- 在沙箱中执行 Agent
- 通过配置和代码自定义 Agent 行为
- 将 KKOCLAW 集成到自己的应用架构中
